Description détaillée

Classe d'accès aux données pour les collections.

Cette classe gère toutes les opérations de lecture et d'écriture des collections, que ce soit depuis la base de données ou l'API TMDB

hasAdultMovies ( int $collectionId)
private

Vérifie si une collection contient des films pour adultes. Cette méthode interroge l'API TMDB pour obtenir les détails d'une collection et vérifie si l'un des films qu'elle contient est marqué comme contenu adulte.

Paramètres
int$collectionIdL'identifiant TMDB de la collection à vérifier
Renvoie
bool Retourne true si la collection contient au moins un film pour adultes, false sinon ou en cas d'erreur de l'API
